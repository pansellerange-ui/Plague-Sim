# Simple Plague Simulation (C#)

This is a beginner-friendly project that simulates the spread of a plague based on:

- Region  
- Year  
- Population  
- Starting infected  

The model is intentionally simple so a high school student can understand it.

## How It Works

The simulation runs for 60 days and updates:

- **Susceptible**
- **Infected**
- **Recovered**

Each day:

Rates automatically change based on year and region.
